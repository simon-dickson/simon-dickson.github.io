What is this?
This is Phase 1 of a structured training roadmap designed to help experienced data analysts transition into AI engineering roles. It bridges the gap between analysis-focused Python and the engineering skills that AI teams actually need.
Who it's for
Data analysts who are comfortable with Python, Pandas, and SQL for ad-hoc analysis and want to level up to writing production-grade, testable, containerised software as a foundation for AI/ML engineering work.
Duration & Approach
Approximately 3 months at 1–2 hours per day on weekdays. Don't aim to finish every course — use them as references. Complete starred (⭐) courses first, then dip into the others for specific topics as you need them.
🎯
Phase 1 Goal
Build a data pipeline that fetches data from a public API, processes it with Pandas, stores it in a SQLite database, and exposes it via a FastAPI endpoint — all tested with pytest and deployed in a Docker container. Push the finished project to GitHub with a proper README.
